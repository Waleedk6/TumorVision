# FYP Project: Code Generator App

A full-stack app for generating Python code based on user instructions.

## Structure
- **backend/**: Flask API (Python).
- **frontend/**: React UI.

## Setup
1. Backend: `cd backend`, `python -m venv venv`, activate venv, `pip install -r requirements.txt`, `python app.py`.
2. Frontend: `cd frontend`, `npm install`, `npm start`.

## Requirements
- Backend: `pip install flask flask-cors` (save to requirements.txt).
- Frontend: Node.js/npm.
